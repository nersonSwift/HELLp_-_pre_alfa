
Перед игрой нужно выбрать Player 1

Для закрытия всех меню 		-> swipe вниз
Для открытия игрового меню 	-> двойной tap в любом месте экрана комнаты
Для перемещения между комнатами -> swipe в направлении движения
Для начала сражения 		-> кнопка в центре экрана 
(Если кнопка не нажимается врагов нет)
(Если движение не возможно, в комнате находится враг препятствующий перемещению)

Управление во время сражения

Переключение врагов 	-> swipe лево <-> право
Переворот карты 	-> swipe вниз
Использование карты 	-> swipe верх
Выбор карты 		-> tap по карте
Окончание битвы 	-> tap по выпавшему кубу

Виды комнат
Обычная 		- Wood,Iron Door
Закрытая 		- CloseDoor (Для входа нужен ключ, первый игрок имеет 2 ключа) 
Комната истезаний 	- DMGDoor (Наносит урон при входе)
